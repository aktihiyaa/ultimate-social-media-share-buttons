== Description ==

== Installation ==

### Standard Installation

1. In your WordPress admin panel, go to **Plugins > Add New**
2. Search for **"Ultimate Social Media Share Buttons"**
3. Click **"Install Now"** and then **"Activate"**

### Manual Installation

1. [Download the plugin](https://downloads.wordpress.org/plugin/ultimate-social-media-share-buttons.zip)
2. Upload to your `/wp-content/plugins/` directory
3. Activate the plugin through the **Plugins** menu in WordPress

### Configuration

1. Go to **Settings > Social Sharing**
2. Configure your preferred sharing options
3. Save changes and view your site to see the buttons

== Frequently Asked Questions ==

= Is this plugin really free? =

Yes! This plugin is completely free with no premium versions, no upsells, and no hidden costs.

= Why should I use this plugin over others? =

This plugin is built for speed. It uses no JavaScript or jQuery, which means it won't slow down your website like many other social sharing plugins. It's pure CSS and HTML for maximum performance.

= Can I choose which social icons to display? =

Absolutely! You can enable or disable each social network individually from the settings page. Only the icons you select will appear on your site.

= Does this plugin have floating/share bar icons? =

Yes! The plugin offers both inline buttons (at the end of your content) and floating sidebar buttons. You can enable either or both options in the settings.

= Does WhatsApp sharing work on desktop? =

Yes! The WhatsApp button works on both desktop and mobile devices. On desktop, it will open WhatsApp Web or prompt you to download the app.

= Will this work with my theme? =

Yes, the plugin is designed to work with all WordPress themes. The buttons use CSS that won't conflict with your theme's styling.

= How do I add nofollow to share links? =

Simply check the "Add Nofollow" option in the plugin settings. This will add `rel="nofollow"` to all share links for SEO purposes.

= Can I hide buttons on mobile devices? =

Yes! You can set a custom viewport width (in pixels) where the floating buttons will hide. The default is 1200px.
